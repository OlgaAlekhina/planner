# API для мобильного приложения "Family Planner" 

<details>
<summary>

## Описание

</summary>

***Мобильное приложение "Family Planner" позволяет создавать календари, списки, заметки, задачи, планировать события, в том числе
регулярные. Основная фишка - возможность совместного использования членами семьи (или группы), делегирование задач.***

</details>

___

<details>
<summary>

## Технологии

</summary>

### REST API
*Django Rest Framework, база данных PostgreSQL*

### Документация
*Swagger*

### Кэширование
*Redis*

### Очереди задач
*Celery, Redis*

### Тестирование
*PyTest*

### Деплой
*Docker Compose, Gunicorn, Nginx*

</details>

<details>
<summary>

## Функционал

</summary>

***1. Методы для работы с пользователями***

| HTTP метод       | URL                     | Описание                              |
|------------------|-------------------------| ------------------------------------------------------ |
| POST             | /api/users/yandex_auth/ | Авторизация пользователей через Яндекс                                               |
| POST             | /api/users/vk_auth/     | Авторизация пользователя через VK ID                                                 |
| POST             | /api/users/mail_auth/   | Авторизация пользователя по адресу электронной почты                                 |
| POST | /api/users/             | Регистрация пользователя с подтверждением через код, высылаемый на электронную почту |
| POST | /api/users/reset_password/ | Восстановление пароля пользователя |
| POST | /api/users/{id}/verify_code/ | Подтверждение кода, высланного на почту |
| GET | /api/users/{id}/ | Получение данных профиля пользователя |
| PATCH | /api/users/{id}/ | Изменение данных профиля пользователя |
| DELETE | /api/users/{id}/ | Удаление профиля пользователя |

***2. Методы для работы с группами (семьями) пользователей***

| HTTP метод       | URL                     | Описание                                                |
|------------------|-------------------------|---------------------------------------------------------|
| POST | /api/groups/ | Создание группы                                         |
| PATCH | /api/groups/{id}/ | Изменение данных группы                                 |
| DELETE | /api/groups/{id}/ | Удаление группы                                         |
| POST | /api/groups/{id}/add_user/ | Добавление участника в группу                           |
| POST | /api/groups/accept_invitation/ | Добавление участника в группу по приглашению            |
| PATCH | /api/groups/{id}/users/{user_id}/ | Изменение данных участника группы                       |
| DELETE | /api/groups/{id}/users/{user_id}/ | Удаление участника из группы                            |
| GET | /api/groups/ | Получение всех групп пользователя                       |
| GET | /api/groups/users | Получение всех групп пользователя со списком участников |
| GET | /api/groups/{id}/ | Получение всех участников группы                        |
| DELETE | /api/groups/{id}/quit_group/ | Выход пользователя из группы                            |

***3. Методы для работы с событиями пользователей***

| HTTP метод | URL                         | Описание                                                |
|------------|-----------------------------|---------------------------------------------------------|
| POST       | /api/events/                | Создание события |
| GET        | /api/events/{id}/           | Получение данных события |
| PATCH      | /api/events/{id}/           | Изменение данных события |
| DELETE     | /api/events/{id}/           | Удаление события |
| GET        | /api/events/                | Получение всех событий пользователя c фильтрацией по дате и поиском по названию |
| DELETE     | /api/events/{id}/quit_event | Отказ пользователя от участия в событии |

***4. Методы для работы с заметками пользователей***

| HTTP метод | URL                         | Описание                                                                        |
|------------|-----------------------------|---------------------------------------------------------------------------------|
| POST       | /api/notes/                 | Создание заметки                                                                |
| GET        | /api/notes/{id}/            | Получение данных заметки                                                        |
| PATCH      | /api/notes/{id}/            | Изменение данных заметки                                                        |
| DELETE     | /api/notes/{id}/            | Удаление заметки                                                                |

***5. Методы для работы с задачами пользователей***

| PATCH      | /api/tasks/{id}/ | Изменение данных задачи |
| DELETE     | /api/tasks/{id}/ | Удаление задачи         |

***6. Методы для работы со списками пользователей***

| HTTP метод | URL                              | Описание                  |
|------------|----------------------------------|---------------------------|

| POST       | /api/lists/{id}/items/           | Создание элемента списка  |

| DELETE     | /api/lists/{id}/items/{item_id}/ | Удаление элемента списка  |
